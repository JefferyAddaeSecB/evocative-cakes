## 📊 Animation Specifications

### Timing
- **Duration:** 0.3s - 40s (context dependent)
- **Easing:** easeInOut, easeOut, linear
- **Delays:** Staggered 0.1-0.3s between elements
- **Repeat:** Infinite for background, once for content

### Performance
- ✅ GPU-accelerated transforms

## 🎯 Customization Guide

### Adjust Animation Speed
```typescript
transition={{ duration: 20 }} // Change to speed up/down
```

### Change Colors
```typescript
animate={{ backgroundPosition: ['...'] }} // Adjust RGB values
```

### Add New Animations
```typescript
<DivMotion
  animate={{ x: [0, 100, 0] }}
  transition={{ duration: 10, repeat: Infinity }}
>
  Your element
</DivMotion>
```
