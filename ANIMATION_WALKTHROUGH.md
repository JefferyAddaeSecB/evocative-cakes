## 📞 Troubleshooting

### If you don't see animations:

1. **Refresh the page** (hard refresh: Cmd+Shift+R)
2. **Check dev server is running:**
   ```bash
   npm run dev
   # Should show "ready in XXX ms"
   ```
3. **Visit localhost:3000** (not 127.0.0.1:3000)
4. **Check browser console** (Cmd+Option+I → Console)
5. **Try different browser** (Chrome, Firefox, Safari)

### If animations are choppy:

1. **Close other tabs** (free up GPU)
2. **Update browser** (latest version)
3. **Check GPU settings** (browser hardware acceleration)
4. **Restart dev server:** `npm run dev`
